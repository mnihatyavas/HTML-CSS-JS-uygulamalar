Yardım Penceresi

Uygulamalarınız hakkında birkaç önemli bilgilendirme:

  1. Gerçekten de güzel bir kayan-pencereli yardım kutusudur.
  2. Bu bilgilendirme kutusu yumuşak kayışını sabit konumlama ve CSS geçiş vasıflarıyla gerçekleştirmektedir.
  3. Kullandığı güzel teknik detaylarına çentikkutulu bilgilendirme sayfasından erişebilirsiniz.
  4. JS kodlaması kullanmanıza gerek kalmadan çentikkutusunun tıklanmasıyla yardım pencereniz "açılacak" ve "kapanacaktır". (Pencereniz enaz IE9'da, yumuşak kayış efektiyse enaz IE10'da çalışmaktadır.)
