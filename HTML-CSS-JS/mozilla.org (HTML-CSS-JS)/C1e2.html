Enformasyon

Bu uygulama konusunda birkaç mühim hatırlatma:

  1. Hakikaten çok zekice tasarlanmış kayarak-görünen bilgi kutusudur.
  2. Bilgi kutusu, görüntüden yumuşak kayış için sabit konumlanmayla css geçişin bileşiminden oluşturulmuştur.
  3. Ayrıca zeki bir teknik olan çentikkutu hilesi kullanılmıştır.
  4. Böylece herhangibir JS kodlaması gerekmeksizin (IE9 ve üzerinde çalışan) hoş bir "tıkla görünsün / tıkla kaybolsun" yumuşak kaydırmalı geçiş efekti yaratılmıştır.
